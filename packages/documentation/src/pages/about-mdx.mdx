#### Inline components
You can create components incline inside your MDX files. Below is an example
of how to do it. You will need to export it for it to work. You can write both
function based components and class based components. The reason why you might
use this is if your component updates the parent state. In the example you need
state to toggle the drop down.

```jsx
// Classed Based Component
export class InlineClassComponent extends React.Component {
  constructor() {
    super()
    this.state = {
      open: false
    }
  }
  render() {
    return (
      <div className="rendered-component">
        <div style={{ backgroundColor: '#112e51' }}>
          <DropDownPanel
            buttonText='Helpdesk'
            cssClass='va-dropdown'
            isOpen={this.state.open}
            contents='Lorem ipsum dolor sit amet, consectetur adipiscing elit. Fusce ullamcorper at eros eu suscipit. Ut imperdiet libero et luctus pretium.'
            clickHandler={() => this.setState({ open: !this.state.open }) }>
            'Lorem ipsum dolor sit amet, consectetur adipiscing elit. Fusce ullamcorper at eros eu suscipit. Ut imperdiet libero et luctus pretium.'
          </DropDownPanel>
        </div>
      </div>
    )
  }
}

<InlineClassComponent />

// Function based component
export const InlinePresentationalComponent = () => (
  <h1>Hello World</h1>
)

<InlinePresentationalComponent />
```

***Gotchas***
- MDX doesn't like newlines in between methods in a class based component.
- Inline components need to begin with an uppercase letter. If it starts with a
  lowercase it will not render.

Bad:

```
export class inlineClassComponent extends React.Component {
  constructor() {
    super()
  }

  render() {
    return (
      <div className="rendered-component">
        Hello World
      </div>
    )
  }
}
```

Good:
```
export class InlineClassComponent extends React.Component {
  constructor() {
    super()
  }
  render() {
    return (
      <div className="rendered-component">
        Hello World
      </div>
    )
  }
}
```
